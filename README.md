# 💬 Java Chat Application using Socket Programming

A real-time multi-client chat application built using **Java**, **TCP Socket Programming**, **Multithreading**, and **Swing GUI**.

---

## 🚀 Features

- Multi-client chat system
- TCP Socket Programming
- Client-Server Architecture
- Java Swing GUI
- Console Client Support
- Public Messaging
- Private Messaging (`@username`)
- Unique Username Validation
- Join & Leave Notifications
- Chat Logging
- Timestamps
- Graceful Disconnect
- Multithreading

---

## 🛠️ Technology Stack

- Java
- Java Swing
- TCP Sockets
- Multithreading
- IntelliJ IDEA

## ▶️ How to Run

### Start the Server

Run:

```
ChatServer.java
```

### Start Client

Run:

```
LoginWindow.java
```

Open multiple client windows to test multi-user communication.
