# Java Socket Programming - Multi-Client Chat Application

## Project Overview

This project implements a Real-Time Multi-Client Chat Application using Java Socket Programming, Multithreading, and Java Swing GUI.

The application supports:

- Multi-client communication
- Public messaging
- Private messaging
- Java Swing graphical user interface
- Chat logging with timestamps
- Duplicate username prevention
- Real-time client-server communication

## Execution

Compile the project:

```bash
javac *.java
```

Run the server:

```bash
java ChatServer
```

Run the client:

```bash
java ChatClient
```

---

## Output

- Server starts successfully
- Multiple clients connect simultaneously
- Public messages are broadcast to all users
- Private messages are delivered to the selected user
- Chat logs are stored with timestamps
